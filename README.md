Project Title: Financial Sales Analysis

Objective: To analyze revenue, cost, and profitability across products, countries, and customer segments.

Tools Used:
- Python
- Pandas
- Matplotlib
- Jupyter Notebook
- Excel (dashboard)

Key Insights:
- Identified top-performing product categories
- Analyzed country-wise revenue contribution
- Evaluated overall business profitability

Outcome:
This project demonstrates financial data analysis, visualization, and business insight generation.
